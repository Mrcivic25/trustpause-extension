# Contributing to TrustPause

We welcome community contributions to improve the TrustPause browser extension!

## Important Note on Architecture
Please note that the **backend API and scoring algorithms are closed source** to protect our proprietary threat intelligence. 
If you are contributing to this repository, you are contributing to the **client-side Manifest V3 extension** only.

- If you notice a bug in the UI, or have improvements for the warning screens, feel free to open a PR!
- If you have an idea for a new client-side heuristic (e.g. better form detection), please open an Issue first to discuss it with the maintainers.
- Do NOT include any API keys, tokens, or credentials in your pull requests.

## How to Submit a Pull Request
1. Fork this repository.
2. Create a new branch (`git checkout -b feature/my-cool-feature`).
3. Make your changes and test them locally by loading the unpacked extension.
4. Commit your changes (`git commit -m "Add my cool feature"`).
5. Push to the branch (`git push origin feature/my-cool-feature`).
6. Open a Pull Request!

Thank you for contributing!
